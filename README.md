# Line Splitter

A modern web application for splitting large text files into smaller chunks based on line count.

## Features

- **Dual Input Methods**: Type directly or drag & drop text files
- **Real-time Processing**: Results update instantly as you type or change settings
- **Line Numbers**: GitHub-style line numbering with click-to-select functionality
- **Flexible Output**: Copy individual chunks or download as separate files
- **Scroll Controls**: Navigate long texts with top/bottom scroll buttons
- **Click-to-Scroll**: Prevent scroll hijacking - click chunks to enable scrolling
- **Copy Tracking**: Visual badges show how many times each chunk was copied
- **Expandable View**: Toggle between split view and full-width results panel

## Usage

1. **Input Text**: Type or drag & drop a `.txt` file into the left panel
2. **Set Chunk Size**: Specify how many lines per chunk
3. **View Results**: Split chunks appear instantly in the right panel
4. **Copy or Download**: Use the action buttons on each chunk

## Technologies

- Pure HTML, CSS, and JavaScript (no frameworks)
- Modern CSS with custom properties
- Responsive design with flexbox

## Local Development

Simply open `index.html` in your browser - no build process required!
